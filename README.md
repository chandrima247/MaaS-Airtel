# MaaS Airtel Insurance Journey

A standalone static prototype for the Airtel Money insurance marketplace journey.

## Pages

- `index.html` - guided home-to-listing journey with AI-style search, calculator, filters, and compare tray.
- `compare.html` - side-by-side plan comparison using selected products and quote context.
- `pdp.html` - J-Junior product detail page with calculator, quote modal, IPF breakdown, and application CTA.

Open `index.html` in a browser to start the journey.
